# Knowledge Base

Сейчас в папке уже есть структурированная база знаний `club-sales-playbook.json`, собранная из документов:

- `Логика продажи.docx`
- `Возражения .docx`
- `Преимущества клуба.docx`
- `Продукт клуба.docx`

Сюда дополнительно можно положить:

- скрипты продаж,
- дерево возражений,
- речевые паттерны для менеджера,
- запрещенные формулировки,
- офферы и тарифные правила,
- кейсы по закрытию на карту.

Следующий шаг: когда ты пришлешь свои файлы, мы подключим их к подсказкам и к post-call разбору.
