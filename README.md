# Sales Souffleur Assistant

Turn-based веб-приложение для менеджера продаж.

## Что делает текущая версия

- слушает разговор с микрофона,
- делит речь на отдельные turns по паузам,
- в live-режиме отправляет каждый turn на diarization,
- старается отличить менеджера от клиента по эталонному голосу менеджера,
- показывает подсказку длиной 100-120 символов только после окончания фразы клиента,
- выводит эту подсказку текстом на экран и озвучивает ее в браузере,
- сохраняет стенограмму и аудио разговора,
- выгружает `.md`, `.json` и аудиозапись.

## Папки

- `index.html` — интерфейс менеджера
- `styles.css` — оформление
- `app.js` — turn detector, запись, UI-логика
- `netlify/functions/coach.js` — diarization и AI-подсказки
- `knowledge-base/` — здесь уже лежит структурированная база знаний, собранная из твоих документов
- `scripts/publish-to-github.ps1` — создание репозитория и загрузка проекта на GitHub через API

## Live-режим

Для live-режима нужны переменные окружения:

- `OPENAI_API_KEY`
- `OPENAI_TRANSCRIBE_MODEL`
- `OPENAI_COACH_MODEL`

По умолчанию:

- diarization: `gpt-4o-transcribe-diarize`
- генерация подсказки: `gpt-4.1-mini`

## Важные ограничения v1

- Для лучшего отделения менеджера от клиента нужно записать эталонный голос менеджера перед звонком.
- Автоматическое различение спикеров в реальном помещении зависит от качества микрофона и шума.
- Demo-режим не делает diarization, он нужен для ручной проверки логики подсказок.

## Публикация на GitHub без git

```powershell
.\scripts\publish-to-github.ps1 -GitHubToken "<TOKEN>" -RepoName "sales-souffleur-assistant"
```
